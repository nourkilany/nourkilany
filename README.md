# nourkilany

Personal profile website built with Astro + TailwindCSS v4.

## Stack

- **Astro** — static site generator
- **TailwindCSS v4** — styling
- **JetBrains Mono** — monospace font

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Static output is generated in `dist/`.
